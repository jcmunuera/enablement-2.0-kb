# Examples

This directory contains before/after examples for each circuit breaker pattern.

## Structure

- **basic-fallback/** - Pattern 1 (most common, 80% cases)
- **fail-fast/** - Pattern 3 (no fallback)
- **multiple-fallbacks/** - Pattern 2 (fallback chain)

Each directory has:
- `before/` - Original code without circuit breaker
- `after/` - Code with circuit breaker applied

## Usage

These examples show realistic transformations. Use as reference for skill execution.
