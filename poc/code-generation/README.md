# PoC: Code Generation

**Estado:** Pendiente de diseño  
**Fecha inicio:** TBD

## Objetivo

Validar que el Knowledge Base estructurado puede alimentar efectivamente un agente AI para generar código que cumpla con los estándares organizacionales.

## Alcance

### Input
- Configuración JSON siguiendo schema de skill-code-020
- Knowledge Base v1.0.0

### Proceso
1. Cargar configuración
2. Skill identifica módulos necesarios
3. AI genera código usando ERIs y MODULEs
4. Validación automática (4 tiers)

### Output Esperado
- Microservicio Java/Spring Boot completo
- Estructura Hexagonal Light
- Patrones de resilience aplicados
- Persistencia (JPA o System API)
- Tests generados
- Validación passing
